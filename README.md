# oasis
Refining ML interatomic potentials with sparse DFT data in low-data regimes.

## Environment

Install the project and test dependencies with:

```bash
pip install -e .[dev]
```

If you use `uv`, the equivalent is:

```bash
uv pip install -e ".[dev]"
```

Run the test suite with:

```bash
python -m pytest
```
